Тема:  Рекурсия
1.	Описать рекурсивную функцию Fact(N) вещественного типа, вычисляющую значение факториала N! = 1∗2∗…∗N (N>0 — параметр целого типа). С помощью этой функции вычислить факториалы пяти данных чисел.
2.	Описать рекурсивную функцию Fib(N) целого типа, вычисляющую N-й элемент последовательности чисел Фибоначчи (N — целое число): F1=F2=1, FK=FK−2+FK−1, K = 3,4,…. Напишите рекурсивную программу, вычисляющую последовательность Фиббоначи для n-чисел. Считать, что номер N не превосходит 20.
3.	Описать рекурсивную функцию CombiN(N, K) целого типа, находящую C(N, K)- число сочетаний из N элементов по K — с помощью рекуррентного соотношения: C(N, 0) = C(N, N) = 1, C(N, K)=C(N—1, K)+C(N—1, K—1) при 0<K<N. Параметры функции — целые числа; N>0, 0≤K≤N. Дано число N и пять различных значений K. Вывести числа C(N, K)вместе с количеством рекурсивных вызовов функции CombiN, потребовавшихся для их нахождения.
Для проверки С^k_n = n!/(n-k)!k!
4.	Описать рекурсивную функцию NOD(A, B) целого типа, находящую наибольший общий делитель (НОД) двух натуральных чисел A и B, используя алгоритм Евклида: НОД(A, B) = НОД(B, A % B), если B≠0;НОД(A, 0)=A. С помощью этой функции найти НОД(A, B), НОД(A, C), НОД(A, D), если даны числа A, B, C, D.
5.	Описать рекурсивную функцию DigitSum(K) целого типа, которая находит сумму цифр целого числа K, не используя оператор цикла.
